API для работы с переводами:
  • get_text(lang, key, **kwargs) - основная функция
  • get_direction_text(lang, direction) - для направлений
  • get_funnel_step_text(lang, step) - для воронки

API для работы с языком в БД:
  • set_user_language(telegram_id, language) - сохранить
  • get_user_language(telegram_id) -> str - получить

Формат переводов:
  TRANSLATIONS = {
      "ru": {"key": "текст"},
      "uz": {"key": "matn"}
  }
