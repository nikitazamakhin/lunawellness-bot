## Как это работает

1. **Первый запуск `/start`:**
   - Пользователь видит выбор языка: 🇷🇺 Русский / 🇺🇿 O'zbekcha
   - После выбора язык сохраняется в БД
   - Все дальнейшие сообщения на выбранном языке

2. **Повторный `/start`:**
   - Сразу показывается меню на сохранённом языке

3. **Воронка:**
   - Все 10 шагов воронки отправляются на языке пользователя

4. **Бронирование:**
   - Весь процесс бронирования на выбранном языке
   - Уведомление админу всегда на русском

## Структура переводов

Все тексты в файле `bot/i18n.py` структурированы по ключам:

```python
TRANSLATIONS = {
    "ru": {
        "welcome": "...",
        "btn_directions": "...",
        # ... и т.д.
    },
    "uz": {
        "welcome": "...",
        "btn_directions": "...",
        # ... и т.д.
    }
}
```

## Добавление новых переводов

Чтобы добавить новый текст:

1. Добавить ключ в `TRANSLATIONS["ru"]` и `TRANSLATIONS["uz"]`
2. Использовать в коде: `get_text(lang, "your_key")`

Пример:
```python
# В i18n.py
TRANSLATIONS = {
    "ru": {
        "new_feature": "Новая функция"
    },
    "uz": {
        "new_feature": "Yangi funksiya"
    }
}

# В коде
lang = await get_user_language(user_id)
text = get_text(lang, "new_feature")
```
