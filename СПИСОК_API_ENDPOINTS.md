## 2️⃣ WEBHOOKS (нужно реализовать)

**Направление:** Бот → Odoo (события в реальном времени)

| # | Событие | Когда срабатывает | Что Odoo делает |
|---|---------|-------------------|-----------------|
| 1 | `user.created` | Новый пользователь запустил бота | Создать лид в CRM |
| 2 | `booking.created` | Пользователь забронировал | Создать сделку |
| 3 | `funnel.step_completed` | Прошёл шаг воронки | Обновить стадию лида |
| 4 | `booking.status_changed` | Статус изменён через API | Обновить в CRM |

**Формат:**
```http
POST https://client-odoo.com/api/luna-webhook
Content-Type: application/json
X-Luna-Signature: sha256=abc123...

{
  "event": "booking.created",
  "timestamp": "2026-05-14T16:15:00Z",
  "data": { ... }
}
```
