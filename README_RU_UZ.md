# LUNA Wellness Bot - Поддержка двух языков (RU/UZ)

## 🎯 Что было сделано

Telegram-бот LUNA Wellness теперь полностью поддерживает **русский** и **узбекский** языки.

### ✅ Реализованные функции

- **Выбор языка** при первом запуске `/start`
- **Сохранение** выбранного языка в базе данных
- **Все сообщения** на выбранном языке (приветствие, меню, направления, бронирование)
- **Воронка продаж** (10 шагов) на выбранном языке
- **Процесс бронирования** полностью на выбранном языке
- **Уведомления администратору** всегда на русском языке

## 🔧 Технические детали

### Структура переводов

Все переводы хранятся в файле `bot/i18n.py`:

```python
TRANSLATIONS = {
    "ru": {
        "welcome": "✨ Добро пожаловать в LUNA...",
        "btn_directions": "🧘‍♀️ Наши направления",
        # ... и т.д.
    },
    "uz": {
        "welcome": "✨ LUNA'ga xush kelibsiz...",
        "btn_directions": "🧘‍♀️ Yo'nalishlarimiz",
        # ... и т.д.
    }
}
```

### API для работы с переводами

```python
from bot.i18n import get_text, get_direction_text, get_funnel_step_text

# Получить текст по ключу
text = get_text(lang, "welcome")

# Получить текст с параметрами
text = get_text(lang, "booking_confirm",
                direction="Reformer Pilates",
                day="Понедельник",
                name="Анна",
                phone="+998 90 123 45 67")

# Получить описание направления
text = get_direction_text(lang, "reformer")

# Получить текст шага воронки
text = get_funnel_step_text(lang, 1)
```

### База данных

В таблицу `users` добавлено поле `language`:

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    telegram_id INTEGER UNIQUE NOT NULL,
    first_name TEXT DEFAULT '',
    username TEXT DEFAULT '',
    language TEXT DEFAULT 'ru',  -- НОВОЕ ПОЛЕ
    funnel_step INTEGER DEFAULT 0,
    booked INTEGER DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Методы для работы с языком:

```python
from bot.database import set_user_language, get_user_language

# Сохранить язык пользователя
await set_user_language(telegram_id, "uz")

# Получить язык пользователя
lang = await get_user_language(telegram_id)  # вернёт "ru" или "uz"
```

## ⚙️ Добавление/изменение переводов

### Добавить новый текст

1. Откройте `bot/i18n.py`
2. Добавьте новый ключ в оба словаря:

```python
TRANSLATIONS = {
    "ru": {
        # ... существующие переводы ...
        "new_message": "Новое сообщение на русском"
    },
    "uz": {
        # ... существующие переводы ...
        "new_message": "Yangi xabar o'zbek tilida"
    }
}
```

3. Используйте в коде:

```python
lang = await get_user_language(user_id)
text = get_text(lang, "new_message")
await message.answer(text)
```

4. Перезапустите бота

### Изменить существующий перевод

1. Откройте `bot/i18n.py`
2. Найдите нужный ключ
3. Измените текст в нужном языке
4. Перезапустите бота

## 🐛 Решение проблем

### Бот не запускается

1. Проверьте, что все файлы скомпилированы:
```bash
python3 -m py_compile bot/i18n.py bot/database.py bot/keyboards.py
```

2. Выполните миграцию БД:
```bash
python3 scripts/migrate_add_language.py
```

### Тексты на неправильном языке

1. Проверьте язык в БД:
```python
lang = await get_user_language(telegram_id)
print(f"User language: {lang}")
```

2. Отправьте `/start` заново для выбора языка
