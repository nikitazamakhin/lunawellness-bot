## Как это работает

### Первый запуск пользователя

1. Пользователь отправляет `/start`
2. Бот показывает выбор языка:
   - 🇷🇺 Русский
   - 🇺🇿 O'zbekcha
3. После выбора:
   - Язык сохраняется в БД
   - Отправляется кружочек #1 (если загружен)
   - Отправляется текст шага 1 на выбранном языке
   - Показывается главное меню на выбранном языке
   - Запускается воронка на выбранном языке

### Повторный `/start`

Сразу показывается главное меню на сохранённом языке.

### Воронка

Все 10 шагов воронки отправляются на языке, который выбрал пользователь.

### Бронирование

Весь процесс бронирования проходит на выбранном языке:
- Выбор направления
- Выбор дня
- Ввод имени
- Ввод телефона
- Подтверждение

Уведомление администратору всегда приходит на русском языке.

После запуска:

1. Отправьте боту `/start` с нового аккаунта
2. Выберите язык (🇷🇺 или 🇺🇿)
3. Убедитесь, что все сообщения на выбранном языке
4. Попробуйте:
   - Открыть разные разделы меню
   - Посмотреть направления

## Добавление/изменение переводов

Все тексты находятся в файле `bot/i18n.py` в словаре `TRANSLATIONS`.

Структура:
```python
TRANSLATIONS = {
    "ru": {
        "key_name": "Текст на русском"
    },
    "uz": {
        "key_name": "Matn o'zbek tilida"
    }
}
```

После изменения переводов — перезапустите бота.

## Команды администратора

Все админ-команды остаются на русском языке:
- `/stats` — статистика
- `/users` — список пользователей
- `/circles` — статус кружочков
- `/setcircle N` — загрузка кружочка
- `/broadcast` — рассылка

## Техническая поддержка

Если что-то не работает:

1. Проверьте логи бота
2. Убедитесь, что миграция выполнена
3. Проверьте, что все файлы обновлены
4. Запустите тесты: `python3 scripts/test_i18n.py`

## Откат изменений

Если нужно вернуться к версии без многоязычности:

1. Верните файл: `mv bot/texts_old.py bot/texts.py`
2. Откатите изменения в handlers через git
3. Перезапустите бота

База данных может остаться с полем `language` — оно не помешает.
